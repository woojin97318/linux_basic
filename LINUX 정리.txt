cd(change directory): 경로 변경
	cd ../../: 한단계 ~ 두단계 상위 폴더로 이동
	cd 폴더명: 폴더명폴더로 이동
	tab: 자동 입력 기능
	cd 전체 경로: 절대 경로
	cd 상대 경로: 자기 자신의 위치를 기준으로 처리

ls: 현재의 리스트
	ls -l: 리스트의 상세정보
	ls ../../: 두단계 상위폴더의 리스트
	ls -a: 숨김 리스트 표시
	ls -al: 복합속성
	ls -R test: test하위를 다 보여줌

pwd: 현재 경로

mkdir 폴더명: 폴더 생성
	mkdir -p a1/a2/a3/a4/a5: a2~5까지 생성해줌

touch 파일명.확장자: 파일 생성

./: 현재 위치

cp ./(1) (2): 1을 2로 복사
	cd -r (1) (2): (1)폴더를 (2)로 복사

cat 파일명.txt: 파일 내용 확인

head 파일명: 파일 10줄까지 보여줌
head -3 파일명: 파일 3줄까지 보여줌

tail 파일명: 파일 마지막 10줄까지 보여줌
tail -3 파일명: 파일 마지막 3줄까지 보여줌

vi 파일명.txt[흑백](vim 파일명.txt[컬러]): 텍스트 편집기
	i - 텍스트 끼워넣기(편집)
	:q! - 저장안하고 나가기
	:w - 변경사항 저장
	:wq - 저장 후 quit
	dd - 한줄 삭제
